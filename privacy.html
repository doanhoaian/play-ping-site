Quick Summary

  • No user accounts required. PlayPing does not collect personal data by default. In certain cases, users may optionally provide an email address (for example, when submitting feedback).
  • An anonymous identifier may be generated locally to support app functionality, analytics, and advertising.
  • Uses Android VpnService to simulate network conditions, primarily processed on-device.
  • Traffic may be temporarily routed through intermediary servers for connectivity purposes; no traffic content is logged.
  • Uses Firebase, advertising partners, and PlayPing backend services for diagnostics and ads.

1. Introduction & Scope

PlayPing (by Dihaver Tech) is an Android application that allows users to simulate network conditions such as packet loss, latency, bandwidth limits, and disconnections for testing and training purposes. This Privacy Policy explains how information is processed when using the app.

2. Data We Process

PlayPing does not require registration and does not collect personal data by default. However, to operate and improve the app, we may process:

  • An anonymous app-generated identifier (not linked to personal identity).
  • Technical data such as crash reports, performance metrics, device model, and Android version.
  • Ad-related technical information required to display ads.
  • An optional email address, if the user chooses to provide it when submitting feedback.

3. Purpose of Processing

  • Ensuring stability, performance, and reliability of the app.
  • Displaying advertisements and managing ad frequency.
  • Responding to user feedback and support requests.

4. Network Simulation & VPN Operation

PlayPing uses Android VpnService to create a simulated network environment.

  • The VPN is used exclusively for simulation purposes.
  • The app does not monitor browsing activity or inspect traffic content.
  • In certain situations, traffic may be temporarily routed through an intermediary server to prevent connectivity loss. These servers do not store, log, or analyze traffic content.

5. App Permissions

PlayPing requests only permissions necessary for its functionality:

Permission Purpose
INTERNET, ACCESS_NETWORK_STATE Network connectivity and status checks.
SYSTEM_ALERT_WINDOW Floating UI elements for quick simulation control.
FOREGROUND_SERVICE, WAKE_LOCK Ensure stable background operation of simulation services.
POST_NOTIFICATIONS, VIBRATE Status updates and user notifications.
